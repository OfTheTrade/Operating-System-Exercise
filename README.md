--------- To Run ---------

Use makefile commands:

make compile : Compiles the executable into build/

make run : Runs the executable (found in build/) with input "01234". Practically join conversation "01234".

make run_alt : Runs the executable (found in build/) with input "56789". Practically join conversation "56789".

make flush: Flushes the shared memory and the semaphore. Should be used if they are non responsive for any reason.
